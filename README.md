# Something Red
忘年会で赤い物が必要なので、とりあえず光ってみる

## プロジェクト構成
- matrix
  - LED制御するFreeRTOSで動くハードウェアのコード
- controller
  - 人が操作するためのコントローラーアプリになる予定

## 注意
- git submoduleを含むので、クローン時にサブモジュールもデータを取得する必要があります
